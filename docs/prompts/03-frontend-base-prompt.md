# Prompt 03 - Frontend Base

Implementa el frontend base del kiosco.

Necesito:
- Next.js app router
- layout portrait/fullscreen kiosk
- design system base
- componentes touch-friendly
- pantalla Home
- bot animado integrado pero secundario
- navegación base entre pantallas
- estado global de sesión

Reglas UX:
- muy poco texto
- botones grandes
- una decisión por pantalla
- estilo premium
