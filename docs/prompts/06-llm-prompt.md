# Prompt 06 - LLM

Implementa la integración LLM según la llm-spec.

Requisitos:
- endpoint aislado
- system prompt fijo
- salida estructurada
- timeouts
- fallback si falla proveedor
- logs seguros
- sin exponer secretos

Casos de uso:
- reformular pregunta
- explicar opción
- acompañar indecisión
- resumir recomendaciones
