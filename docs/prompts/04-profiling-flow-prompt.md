# Prompt 04 - Profiling Flow

Implementa el flujo MVP de perfilamiento según la UX spec.

Pantallas:
- tipo propiedad
- objetivo compra
- ubicación
- rango de presupuesto/dividendo
- subsidio
- cierre de perfil

Requisitos:
- guardar cada respuesta
- permitir saltar
- permitir “no estoy seguro”
- mostrar progreso
- no perder sesión
